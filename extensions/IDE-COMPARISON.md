# GuardRail AI - IDE Comparison Guide

## Universal Compatibility

GuardRail AI extension works identically across all VS Code-based IDEs!

## Supported IDEs

### ✅ Fully Supported (VS Code-based)

| IDE | Status | Installation | Notes |
|-----|--------|--------------|-------|
| **Kiro** | ✅ Full Support | `./install-kiro.sh` | AI-powered IDE, full compatibility |
| **Cursor** | ✅ Full Support | `./install.sh` | AI-first IDE, works alongside Cursor AI |
| **Windsurf** | ✅ Full Support | `./install.sh` | Formerly Anity Gravity, full support |
| **VS Code** | ✅ Full Support | `./install.sh` | Original, reference implementation |
| **VS Codium** | ✅ Full Support | `./install.sh` | Open-source VS Code, full support |
| **Code - OSS** | ✅ Full Support | `./install.sh` | VS Code open source, full support |

### 🚧 Requires Different Implementation

| IDE | Status | Implementation Needed |
|-----|--------|----------------------|
| IntelliJ IDEA | ❌ Not Supported | Kotlin/Java plugin required |
| PyCharm | ❌ Not Supported | Kotlin/Java plugin required |
| WebStorm | ❌ Not Supported | Kotlin/Java plugin required |
| Sublime Text | ❌ Not Supported | Python package required |
| Vim/Neovim | ❌ Not Supported | Lua/VimScript plugin required |
| Emacs | ❌ Not Supported | Elisp package required |
